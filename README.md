# ModrinthDownloads

> A TUI to see a live Modrinth download count.

## About

Ever been in your terminal on your computer, then decided you want to see someone's download count on Modrinth?
You probably havn't ever had that happen, but if you get this package, then you don't need to worry if it does!

## Install

Install command:

`pip install modrinthdownloads`
